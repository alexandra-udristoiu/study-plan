# Screenshots

Drop screenshot PNGs here using these exact filenames so the main README picks them up:

| File | Screen |
|------|--------|
| `notes.PNG` | Notes list organised by topic |
| `flashcard.PNG` | Flashcard review session |
| `studynote.PNG` | An individual study note |

Tips for capturing on an emulator/device:

```bash
adb exec-out screencap -p > screenshots/notes.png
```

Keep images portrait, ideally the same dimensions, so they line up in the README table.
